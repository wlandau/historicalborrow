# Plot tau

Plot the tau parameter of a fitted hierarchical model.

## Usage

``` r
hb_plot_tau(mcmc)
```

## Arguments

- mcmc:

  Data frame of posterior samples generated by
  [`hb_mcmc_hierarchical()`](https://wlandau.github.io/historicalborrow/reference/hb_mcmc_hierarchical.md).

## Value

A `ggplot` object

## See also

Other plot:
[`hb_plot_borrow()`](https://wlandau.github.io/historicalborrow/reference/hb_plot_borrow.md),
[`hb_plot_group()`](https://wlandau.github.io/historicalborrow/reference/hb_plot_group.md)

## Examples

``` r
data <- hb_sim_independent(n_continuous = 2)$data
mcmc <- hb_mcmc_hierarchical(
  data,
  n_chains = 1,
  n_adapt = 100,
  n_warmup = 100,
  n_iterations = 200
)
hb_plot_tau(mcmc = mcmc)
```
